How To Use

This weather app is a simple tool that allows you to check the current weather conditions in any city in the world. To use the app, simply enter the name of the city in the search bar and press enter. The app will then display the current temperature, humidity, wind speed, and other weather conditions for that city.

The app uses the Rapid API to get weather data. The Rapid API is a free API that provides weather data for cities all over the world.

Features

  • Check the current weather conditions in any city in the world
  • Get weather data for the current day

Getting Started

To get started with the weather app, simply download the latest release from GitHub. Once you have downloaded the app, unzip the file and double-click on the `weather.app` file to launch the app.

When you first launch the app, you will be prompted to enter a city name. Enter the name of the city that you want to check the weather for and press enter. The app will then display the current weather conditions for that city.

You can also use the app to get weather data for the next 7 days. To do this, click on the `7 day forecast` button. The app will then display a table showing the weather conditions for each day of the week.

The app also provides weather alerts for your location. To enable weather alerts, click on the `settings` button and then select the `enable weather alerts` checkbox. The app will then send you notifications when there are weather events in your area.
