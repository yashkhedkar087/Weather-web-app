About the Weather App

This weather webapp is a simple tool that allows you to check the current weather conditions in any city in the world. To use the app, simply enter the name of the city in the search bar and press enter. The app will then display the current temperature, humidity, wind speed, and other weather conditions for that city.

The app uses the API Ninjas to get weather data. The API Ninjas is a free API that provides weather data for cities all over the world.

The app is open source and can be found on GitHub. If you would like to contribute to the app, please feel free to fork the project and submit a pull request.

Features

  • Check the current weather conditions in any city in the world
  • Get weather data for the current day
  • Get weather alerts for your location
  • Set up notifications for weather events
